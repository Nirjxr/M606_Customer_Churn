# Telco Customer Churn Prediction Pipeline

## 📌 Project Overview
In the telecommunications industry, retaining existing customers is significantly more cost-effective than acquiring new ones. This project focuses on building an **end-to-end Machine Learning pipeline** to predict customer churn (whether a customer will stay or leave).

Using the **Telco Customer Churn dataset**, we perform data cleaning, exploratory data analysis (EDA), feature engineering, and model training to identify "at-risk" customers. The final solution provides actionable business insights to improve retention strategies.

## 📂 Dataset Details
* **Dataset Name:** Telco Customer Churn
* **Source:** [Kaggle / IBM Sample Data Sets]
* **Rows:** 7,043 customers
* **Columns:** 21 features (Demographics, Services, Account Info)
* **Target Variable:** `Churn` (Yes/No)

## 🛠️ Technologies & Libraries
The project is implemented in **Python** using a Jupyter Notebook environment (Google Colab).
* **Data Manipulation:** `pandas`, `numpy`
* **Visualization:** `matplotlib`, `seaborn`
* **Machine Learning:** `scikit-learn` (Logistic Regression, Random Forest)
